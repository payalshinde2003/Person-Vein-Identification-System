# Person-Vein-Identification-System
This project implements a Finger Vein Recognition System using Machine Learning and Deep Learning algorithms. The system uses image processing techniques to extract vein pattern information from finger vein images and compares the performance of SVM and CNN models for accurate biometric identification.
# Finger Vein Recognition System

This project is developed using Machine Learning and Deep Learning techniques for biometric authentication using finger vein images.

## Technologies Used
- Python
- OpenCV
- TensorFlow
- CNN
- SVM
- Tkinter

## Features
- Dataset preprocessing
- CNN and SVM training
- Vein image prediction
- Accuracy graph
- GUI application

## How to Run

1. Upload dataset
2. Click Preprocess
3. Run SVM and CNN
4. Click Predict
